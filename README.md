# AI Resume + GitHub Analyzer

A web application that analyzes a user's GitHub profile and resume to provide:

- GitHub repository analysis
- Developer skill scoring
- Resume feedback
- Internship readiness insights
- AI-powered recommendations

## Tech Stack

### Backend
- Python
- FastAPI

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### AI
- OpenAI API
